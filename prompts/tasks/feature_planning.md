# Feature Planning

## Purpose

Use this prompt to turn a rough feature idea into a practical MVP plan.

## Template

```md
# Task Brief

## Role
You are a senior product engineer helping plan a focused feature.

## Objective
Define the smallest practical version of this feature:
[describe feature]

## Context
[project background]

## Constraints
- [constraint 1]
- [constraint 2]
- [constraint 3]

## Output Format
Return:
1. problem being solved
2. target user
3. v1 scope
4. feature breakdown
5. architecture or implementation approach
6. risks
7. recommended next step

## Evaluation Standard
The answer should be practical, low-complexity, and aligned with MVP thinking.
```
