# Prompt Template

Every serious prompt should include these six parts.

```md
# Prompt Template

## Role
Who the AI is for this task.

## Objective
What exact outcome is needed.

## Context
Relevant project background and current state.

## Constraints
Limits, rules, non-goals, and boundaries.

## Output Format
The exact structure the response should follow.

## Evaluation Standard
How the output will be judged.
```

## Why this works

- It reduces vague prompting.
- It packages context cleanly.
- It makes constraints visible.
- It creates outputs that are easier to compare and revise.
