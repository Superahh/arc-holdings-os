# Experiment 001

## Name

Baseline prompt vs structured prompt

## Question

Does a six-part prompt template produce more practical outputs than a loose one-line request?

## Setup

- use the same task for both runs
- keep the model constant
- compare output quality against `specs/success_criteria.md`

## What to observe

- clarity
- actionability
- scope discipline
- usefulness of the output structure

## Expected learning

The structured prompt should outperform the loose prompt in usefulness and revision quality.
