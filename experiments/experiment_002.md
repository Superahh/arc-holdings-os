# Experiment 002

## Name

Constraint-light vs constraint-heavy prompt

## Question

How much do explicit constraints improve relevance and reduce unnecessary complexity?

## Setup

- use the same task
- keep role and objective constant
- vary only the constraint section

## What to observe

- complexity of the solution
- alignment with version 1 scope
- amount of rework needed after the first response

## Expected learning

Explicit constraints should reduce overbuilding and make outputs easier to implement.
