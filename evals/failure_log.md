# Failure Log

Use this file to log prompt failures and revisions.

## Prompt run log template

```md
## Prompt name
[name]

## Date
[YYYY-MM-DD]

## Model used
[model]

## Goal
[goal]

## Prompt version
[version]

## Output summary
[summary]

## What worked
- [point]

## What failed
- [point]

## What was missing
- [point]

## Revision for next run
[change]
```

## Reminder

Do not log only that something felt "off."
Log what specifically failed and what should change next.
